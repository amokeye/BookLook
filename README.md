# BookLook

## Description
*Book Look* is a dynamic application that allows the user to search for book via search engine. Book Look uses Mongoose, Apollo imports, and react to provide the user with an efficient experience. The user is responsible for typing any word from a book title into the search bar. The user is then presented with a list of books to choose from.
